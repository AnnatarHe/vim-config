## Vim 配置

这是我个人的 vim 配置文件。

进入 vim 安装：

```
:PlugInstall
```

## 其他

推荐安装 [Fria Code Font](https://github.com/tonsky/FiraCode) 作为编程字体。它有很多漂亮的符号

主题样式是 Ayu Theme

请首先安装 [fzf](https://github.com/junegunn/fzf) 和 [ag](https://github.com/ggreer/the_silver_searcher)

## short keys

`;t` 展开/关闭 目录,

`;cc` 注释选中的文字.

`;cu` 取消注释

`;[space]` 搜索关键字取消高亮.

`;y` 复制进入系统剪切板

`;p` 从系统剪贴板粘贴

`Ctrl-p` 文件名搜索

`;f` 全局搜索
