#  short keys

`;t` is toggle NERDTree,

`;cc` is comment the code that was blcoked.

`;cu` is uncomment the code

`;[space]` is unhighlight the result be search.

`;y` 复制到系统剪切板

`;p` 从系统剪切板粘贴内容
